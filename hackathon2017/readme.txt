Instalación

Si no se tiene instalado nodejs:
Para correr en Ubuntu 16.04

curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash -
sudo apt-get install -y nodejs
sudo apt-get install npm


git clone https://github.com/tamaraOrtiz/hackathon2017.git
cd hackathon2017/proyectandopy
sudo npm install -g cordova ionic
npm install --save 

api:
entrar a la carpeta proyectandoapi	
	Python 3.5.2
pip o pip3 en caso de tener varias versiones de Python
Django 1.8: pip3 install Django==1.8
	pip3 install -r requirement.txt

	
		
	

Para ejecutar la aplicación

Abrir la terminal en la carpeta del proyecto y ejecutar

ionic serve

Para ejecutar la API

Abrir la terminal en la carpeta del proyecto y ejecutar

python manage.py run server
